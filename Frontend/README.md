# Product Showcase - Frontend

## Project Overview
This directory contains the client-side application for the Product Showcase GVCC project. It provides a user interface to display products, filter items, and interact with the backend API.

## Tech Stack
* **Framework:** React.js
* **Styling:**  Tailwind CSS V4
* **HTTP Client:** Axios
* **Build Tool:** Vite / Webpack

## Prerequisites
* Node.js (v14 or higher)
* pnpm (Performant Node Package Manager)

## Installation & Setup

1.  **Navigate to the frontend directory:**
    ```bash
    cd frontend
    ```

2.  **Install dependencies:**
    ```bash
    pnpm install
    ```

3.  **Run the application:**
    ```bash
    pnpm dev
    ```

## Application Structure
* `src/components`: Reusable UI components
* `src/pages`: Main application pages
* `src/assets`: Static images
* `src/utils`: Protected route located in here
* `src/ccontexts`: Auth and Theme context
* `src/Layouts`: Side bar Layout
* `src/lib`: Axios Api instance
* `src/utils`: Protected route located in here
